# QuorinOS

An open source Android ROM based on LineageOS, rebuilt as a pocket Linux:
transparency, user control, minimalism, and a strong visual identity.

The core idea: one phone, many faces. A device that shows different things
depending on who is holding it.

This repo only holds the Quorin overlays, product configs, theming and custom
pieces. It is not a copy of the LineageOS sources, which are fetched separately
with repo.
